## 安装 MAVROS (for Noetic)

```
# 安装 mavros,前提是下载ROS Noetic
sudo apt-get install ros-noetic-mavros

# 下载 MAVROS 依赖的地理数据集
cd /opt/ros/noetic/lib/mavros
sudo ./install_geographiclib_datasets.sh
```

[具体如何使用请自行查询](https://wiki.ros.org/mavros)
[github官方仓库](https://github.com/mavlink/mavros)
